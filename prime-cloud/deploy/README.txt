Deployed by your Senior Engineer.

HOW TO DEPLOY ON HOSTINGER NODE.JS HOSTING:
1. Upload all files via cPanel File Manager or FTP
2. Go to "Node.js" section in cPanel
3. Set Node.js version to 18 or higher
4. Set Application Root to the uploaded directory
5. Set Application Entry Point to: server.js
6. Set Environment Variables (or use the .env file)
7. Start the application

ACCESS URLs:
- Main Dashboard: http://your-domain.com/
- POS System: http://your-domain.com/pos
- API: http://your-domain.com/api/v1
